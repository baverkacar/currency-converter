# Currency Converter
This project is a static website made using flask framework. For converting currency fixer.io has been used.
## Required Tools

### Flask

For download flask:

```bash 
  $ pip install Flask
```
For run app:

```bash 
  $ FLASK_APP=app.py flask run 
```
